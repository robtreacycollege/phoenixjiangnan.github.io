Priciple for Object Parsing from Json

The json parsing priciple that I learned from the above experience is:

  1. every pair of curl brace {}, with or withour name, should be mapped to a class
  2. every pair of square bracket [], with or withour name, should be mapped to a list of objects
  3. field name in class should match exactly the json field name, but class name can be different
